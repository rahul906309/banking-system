# Bank Management System

## Project Overview

The Bank Management System is a desktop application developed using Python and Tkinter. The application simulates basic banking operations, allowing users to create accounts, deposit money, withdraw funds, check balances, and manage customer account information.

Customer data is stored in a text file, making it suitable for learning basic file handling concepts and GUI development using Python.

---

## Objectives

- Create customer accounts.
- Perform banking transactions.
- Manage account information.
- Store customer records.
- Learn Python GUI programming.
